***# Project Description***



This project is part of the Module 8 Git and GitHub practical assignment.



The project demonstrates the use of Git and GitHub in a local development workflow.



\## Objectives



The main objectives of this project are:



\- Create a public GitHub repository

\- Initialize Git locally

\- Create a proper folder structure

\- Make multiple commits

\- Work with branches

\- Merge branches into the main branch

\- Push the project to GitHub
